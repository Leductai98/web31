GIỚI THIỆU CHUNG

GIÁO TRÌNH

  • Hệ thống giáo trình bao gồm : Slide, video, code mẫu chi tiết giúp bạn chủ động tìm hiểu bài trước và ôn tập các kiến thức.
  • Hệ thống học LMS giúp bạn truy cập giáo trình mọi lúc mọi nơi, bạn chỉ cần đăng nhập là có thể học tiếp. Quyền sở hữu giáo trình vĩnh viễn.

NỘI DUNG CHƯƠNG TRÌNH

Giới thiệu tổng quan khóa học, các khái niệm, công cụ, công nghệ web phổ biến. Lắng nghe bài toán bảo mật ứng dụng web của doanh nghiệp

Injection(SQL, NoSQL Injection, OS Command, SSTI)

Broken Authentication

Sensitive Data Exposure

XML External Entities (XXE)

Using Components with Known Vulnerabilities, Insufficient Logging & Monitoring

Broken Access Control

Security Misconfiguration

Cross-Site Scripting (XSS)

Insecure Deserialization
